# Mini RAG Testing Project

This project demonstrates a simple Retrieval-Augmented Generation (RAG) chatbot built using Python, FAISS, Sentence Transformers, and Gemini API.

## Features
- document-based retrieval
- prompt grounding
- hallucination testing
- QA test prompts
- automated evaluation

## Learning Outcome
